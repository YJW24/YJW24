                # 弹性布局

                弹性布局是一种一维布局，一次只能处理一个维度上的元素布局。通过设置display: flex创建 flex 容器，设置之后属性都有一些初值：

                • 元素排为一行flex-direction: row；
                • justify-content: flex-start从主轴起始线开始，主轴方向不会被拉伸（元素宽度是多宽在主轴上就是多少，不像块级元素占满一行了），但可被压缩（元素太多超出容器不会换行）；
                • align-items属性为stretch在交叉轴方向就被拉伸；
                • flex-wrap属性为nowrap；
                • flex-basis属性为auto；

                # 弹性布局中控制对齐方式

                • justify-content主轴元素排列方式。
                • align-items交叉轴上所有元素对齐方式。
                • align-self交叉轴上单个元素对齐方式(给元素设置)。
                • align-content多条主轴的元素在交叉轴的对齐方式。

                # 弹性布局中控制排序方式

                • flex-direction控制主轴方向；
                • order

                # flex-basis

                flex-basis属性在任何空间分配之前初始化弹性元素尺寸，默认值为 auto。如果flex-basis设置为 auto 浏览器会先检查元素是否设置了宽度，在计算他们的初始值进行空间分配；

                当设置为flex-basis:auto，弹性盒子将元素的max-content大小作为flex-basis。

                在空间分配时如果设置了flex-basis: 0，弹性盒子将可以抢占所有空间，并按比例进行分配。

                # flex-grow

                flex-grow属性指定了弹性增长因子，决定了在分配正可用空间时，弹性元素相对于弹性容器中其余弹性元素的增长程度。查看示例代码对flex-grow和flex-basis进行解释说明。

                # flex-shrink

                flex-shrink属性指定了弹性收缩因子，决定了在分配负可用空间时，弹性元素相对于弹性容器中其余弹性元素收缩的程度。查看示例代码对flex-shrink和flex-basis进行解释说明。

                # 总结

                什么设置了元素基本大小？

                1. flex-basis: auto，且元素设置了宽度 => 元素大小为设置的宽度；
                2. flex-basis: auto / content => 元素大小为原始大小；
                3. flex-basis不为0 => 元素的大小为设置的flex-basis值；
                4. flex-basis: 0 => 元素的大小在空间分配时计算。

                当所有元素相加，总宽度小于容器宽度，则有正空间，并且flex-grow会生效；
                当所有元素相加，总宽度大于容器宽度，则有负空间，并且flex-shrink会生效；

                <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>flex-grow 和 flex-basis</title>
    <style>
      html,
      body {
        height: 100%;
      }

      .box {
        display: flex;
        margin-bottom: 48px;
        width: 500px;
        border: 2px dotted rgb(96, 139, 168);
      }

      .box > * {
        border: 2px solid rgb(96, 139, 168);
        border-radius: 5px;
        background-color: rgba(96, 139, 168, 0.2);
      }

      .flex00auto {
        flex: 0 0 auto;
      }

      .flexautoBox > * {
        flex: 1 1 auto;
      }

      .flex0Box > * {
        flex: 1 1 0;
      }
    </style>
  </head>

  <body>
    <!-- 采用 flex: 0 0 auto -->
    <!-- 元素为原始大小 -->
    <div class="box flex00auto">
      <div>One</div>
      <div>Two</div>
      <div>Three has more content</div>
    </div>

    <!-- 采用 flex: 1 1 auto -->
    <!-- 
    设置 flex-basis 值为auto，并且不设置宽度，会自动调整大小。
    弹性盒子的大小取决于元素的 max-content。
    在布局结束后还有 正可用空间 ，会在每个元素之间平均分配，比较大的元素会分到的更多。
   -->
    <div class="box flexautoBox">
      <div>One</div>
      <div>Two</div>
      <div>Three has more content</div>
    </div>

    <!-- 采用 flex: 1 1 0 -->
    <!-- 
    设置 flex-basis 值为0，并且不设置宽度，元素尺寸计算值是0，所有空间都用来争夺。
    设置了相同 flex-grow 值，每个都获得相等空间分配。
   -->
    <div class="box flex0Box">
      <div>One</div>
      <div>Two</div>
      <div>Three has more content</div>
    </div>
  </body>
</html>
                basis 和 shrink
                <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>flex shrink 和 flex basis</title>
    <style>
      html,
      body {
        height: 100%;
      }

      .box {
        display: flex;
        margin-bottom: 48px;
        width: 500px;
        border: 2px dotted rgb(96, 139, 168);
      }

      .box > * {
        border: 2px solid rgb(96, 139, 168);
        border-radius: 5px;
        background-color: rgba(96, 139, 168, 0.2);
      }

      .flexshrink0 > * {
        flex: 0 0 auto;
        width: 200px;
      }

      .flexshrink1 > * {
        flex: 0 1 auto;
        width: 200px;
      }

      .flexautoBox > * {
        flex: 1 1 auto;
      }
    </style>
  </head>

  <body>
    <!-- 设置flex-shrink: 0 -->
    <!-- flex-shrink: 0 不允许收缩，从而溢出了盒子 -->
    <div class="box flexshrink0">
      <div>One</div>
      <div>Two</div>
      <div>Three has more content</div>
    </div>

    <!-- flex-shrink: 1 允许元素收缩，每个元素收缩了同样大小的量。 -->
    <div class="box flexshrink1">
      <div>One</div>
      <div>Two</div>
      <div>Three has more content</div>
    </div>

    <!-- 
    首先设置了 flex-basis: auto，会自动调整大小。
    又设置了flex-shrink 允许元素收缩，这些元素会以min-content的大小进行铺设，会利用自动换行机会来改变。
   -->
    <div class="box flexautoBox">
      <div>Item One</div>
      <div>Item Two</div>
      <div>Item Three has more content and so has a larger size</div>
    </div>
  </body>
</html>
